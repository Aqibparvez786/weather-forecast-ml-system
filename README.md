# Weather Forecast ML System

An end-to-end machine learning weather forecasting system using:

- OpenWeatherMap API
- Automated cron-based data collection
- Exploratory Data Analysis (EDA)
- Random Forest forecasting model
- Django deployment
- MLOps integration

## Features

- Automated hourly weather data collection
- Multi-city weather tracking
- CSV-based historical storage
- Data visualization and profiling
- Time-series forecasting pipeline (in progress)

## Tech Stack

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- OpenWeatherMap API
- Linux Cron Jobs
- Conda
- VS Code
